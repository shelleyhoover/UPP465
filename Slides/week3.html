Refresher: Basemaps v. feature layer maps

There are two primary spatial data formats used in GIS:

  • Raster
  • Vector
  • Not always, but for the sake of simplicity, basemaps are usually rasters and feature layers are usually vectors.
  • We are concerned with manipulating and adding feature layers in this course.

Common Vector Data Types

  • Esri Shapefile
  • Geographic JavaScript Object Notation (GeoJSON)
  • Google Keyhole Markup Language (KML/KMZ)
  • OpenStreetMap OSM XML

A more comprehensive list if you're curious

Vectors are composed of:

  • Points
  • Lines
  • Polygons

Points, lines, and polygons represent the geometry type

Additional attributes are recorded for each feature

Attribute types

  • Integer:
    • non-fractional numbers
    • 0,1,2,8009, etc
  • Real:
    • fractional numbers
    • 0.13, 2.5, 3.0, etc
  • String:
    • a “string” of characters
    • “My dog is barking”, “0134”, “I have 17 dogs”

Note that numerical values can be stored as strings
